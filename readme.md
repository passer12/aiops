# github仓库智能运维系统

2024 NKU 软件工程 课程项目

使用Django + Vue + MySQL 实现

Github项目地址https://github.com/passer12/aiops

## 运行项目

### 本地部署

#### 前端

进入sakai-vue前端文件夹

```
npm install

npm run  dev
```

#### 后端

进入backend文件夹, 配置好数据库后运行

```
python manage.py  makemigration

python manage.py migrate

python manage.py runserver
```
